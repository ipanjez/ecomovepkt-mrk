🎮 Panduan Bermain

🎯 Cara Main: Mode Ranked

  • Masukkan nama, pilih level, lalu tekan Mulai.
  • Tebak kategori sampah yang muncul: Sisa Makanan, Laku Jual, atau Residu.
  • Kamu bisa memilih jawaban dengan klik atau dengan drag kartu sampah ke opsi yang tepat.
  • Salah = waktu berkurang 5 detik & combo reset.

📚 Mode Kurikulum

Latihan fokus per kategori. Pilih SISA MAKANAN, LAKU JUAL, atau RESIDU sebagai fokus. Pemain akan mendapat sampah acak dari semua kategori dan harus memutuskan apakah sampah tersebut SESUAI dengan fokus yang dipilih (2 pilihan: Sesuai / Tidak Sesuai).

Kamu dapat menjawab dengan menekan tombol atau menyeret kartu sampah ke pilihan ✅ Sesuai atau ❌ Tidak Sesuai. Daily challenge tidak aktif di mode Kurikulum, dan badge Top 10 hanya berlaku di Mode Ranked.

⚡ Combo & Poin

Jawab benar berturut-turut → combo naik (max x3 multiplier). Bumi Lestari: +10pt/jawaban. Eko Aktif: +15pt. Penjaga Bumi: +20pt. Semua dikali multiplier combo.

🏅 Badge Sistem

Badge Utama dihitung dari nilai komposit (c) sebagai berikut:

  • Skor: dibagi 13 lalu dikalikan 0.5 (maks. 50 poin).
  • Akurasi: dikalikan 0.3 (maks. 30 poin).
  • Combo tertinggi: dikalikan 4 lalu 0.15 (maks. 100 poin jika combo = 25).
  • Jumlah percobaan: memberi bonus kecil hingga +5 poin.

Rentang nilai badge utama:

  • Bronze: c < 30
  • Silver: 30 ≤ c < 45
  • Gold: 45 ≤ c < 58
  • Platinum: 58 ≤ c < 70
  • Epic: 70 ≤ c < 80
  • Grandmaster: 80 ≤ c < 88
  • Diamond: 88 ≤ c < 93
  • Legend: 93 ≤ c < 96
  • Master: 96 ≤ c < 99
  • Immortal: c ≥ 99

Contoh perhitungan badge utama:

  • Bronze: skor 0, akurasi 0%, combo 0, percobaan 0 → c = 0
  • Silver: skor 0, akurasi 35%, combo 25, percobaan 100 → c ≈ 30.5
  • Gold: skor 0, akurasi 85%, combo 25, percobaan 100 → c ≈ 45.5
  • Platinum: skor 225, akurasi 100%, combo 24, percobaan 100 → c ≈ 58.1
  • Epic: skor 525, akurasi 100%, combo 25, percobaan 100 → c ≈ 70.2
  • Grandmaster: skor 800, akurasi 100%, combo 24, percobaan 100 → c ≈ 80.2
  • Diamond: skor 1000, akurasi 100%, combo 25, percobaan 100 → c ≈ 88.5
  • Legend: skor 1125, akurasi 100%, combo 25, percobaan 100 → c ≈ 93.3
  • Master: skor 1200, akurasi 100%, combo 25, percobaan 100 → c ≈ 96.2
  • Immortal: skor 1275, akurasi 100%, combo 25, percobaan 100 → c ≈ 99.0

Badge Bonus dapat muncul bersamaan dengan badge utama. Kriteria utama:

  • Top 10: berada dalam peringkat 10 besar leaderboard mode ranked (tidak berlaku untuk Mode Kurikulum).
  • Versatile: akurasi setiap kategori seimbang, selisih antar-kategori ≤15%, dan minimal 70% di semua kategori.
  • Rising Star: skor sesi ini naik ≥75 poin dibanding personal best sebelumnya.
  • Daily Challenger: menyelesaikan target daily challenge hari ini.
  • Accuracy Ace: akurasi ≥95% dengan minimal 20 jawaban benar dalam satu sesi.
  • Category Master: 100% benar di satu kategori dengan minimal 5 jawaban di kategori tersebut.

Tags memberikan label performa tambahan pada hasil kamu:

  • #ResiduExpert: akurasi ≥95% pada kategori RESIDU.
  • #SisaMakananExpert: akurasi ≥95% pada kategori SISA MAKANAN.
  • #LayakJualExpert: akurasi ≥95% pada kategori LAKU JUAL.
  • #ComboPro: mencapai combo x25 atau lebih.
  • #Grinder: 100x percobaan / 100x bermain.
  • #GodAim: meraih 100% akurasi.
  • #Persistence: sudah bermain 50+ kali tanpa menempati posisi 1.
  • #WeekWarrior: menyelesaikan 7+ daily challenge.
  • #MonthlyMaster: menyelesaikan 30+ daily challenge.

🔥 Daily Challenge & Streak

Setiap hari ada tantangan unik (skor target, akurasi, kategori fokus). Selesaikan untuk dapat badge Daily Challenger dan tambah streak harian. Daily challenge hanya berlaku di Mode Ranked; tidak muncul di Mode Kurikulum. Streak putus jika tidak selesai challenge sehari penuh.

🏆 Leaderboard

Global real-time. Hanya skor tertinggi yang disimpan per nama. Mode Ranked memiliki Top 10 dan daily challenge, sedangkan Mode Kurikulum fokus pada perbaikan kategori tanpa badge Top 10.
