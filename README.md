# 🌍 EcoMove: Tantangan Memilah Sampah ♻️

Selamat datang di **EcoMove**! Sebuah permainan interaktif berbasis web yang seru dan edukatif. Di sini, Anda ditantang untuk belajar bagaimana memilah berbagai jenis sampah dengan benar demi menjaga kelestarian bumi kita. 🌱

Proyek ini dilengkapi dengan papan peringkat (*leaderboard*) berbasis *cloud* secara *real-time* yang ditenagai oleh Firebase Realtime Database. Mari bermain, kumpulkan poin tertinggi, dan jadilah pahlawan lingkungan! 🏆

---

## ✨ Fitur Unggulan

- 🎮 **Permainan Interaktif & Edukatif:** Belajar memilah sampah organik, anorganik, dan berbahaya (B3) dengan cara yang menyenangkan.
- ☁️ **Leaderboard Real-time:** Bersaing skor dengan pemain lain secara langsung menggunakan integrasi Firebase.
- 📱 **Desain Responsif & Modern:** Tampilan antarmuka yang bersih, menarik, dan responsif, dibangun dengan keandalan **Tailwind CSS**.

---

## 🚀 Cara Penggunaan & Instalasi

Ikuti langkah-langkah mudah berikut untuk menjalankan proyek ini di perangkat Anda:

1. **Clone repositori ini** ke direktori lokal Anda:
   ```bash
   git clone https://github.com/ipanjez/ecomovepkt-mrk.git
   ```
2. **Atur Konfigurasi Firebase:** 
   Buka file `index.html` di kode editor pilihan Anda, kemudian cari bagian `firebaseConfig`. Masukkan kredensial konfigurasi Firebase proyek Anda pada bagian tersebut agar *leaderboard* dapat berfungsi dengan baik.
3. **Mulai Bermain:**
   Cukup buka file `index.html` menggunakan *web browser* (seperti Google Chrome, Mozilla Firefox, atau Safari) untuk langsung memainkannya!
